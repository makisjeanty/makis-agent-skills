## Delivery standards

- Keep controllers thin, services focused, repositories dedicated.
- Validate all input, escape all output, protect state-changing actions.
- Write the test before the fix for bugs and security-sensitive changes.
- Capture reusable patterns in `makis-digital-memory` after meaningful work.
- Run `makis-digital-verification-loop` before closing any task.
